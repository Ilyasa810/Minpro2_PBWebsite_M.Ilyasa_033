# Portofolio Website

## Profil
- **Nama**: Muhammad Ilyasa' 'Izzuddin
- **Project**: Portofolio Website

## Fitur
1. **Home Section** - Perkenalan singkat dengan nama dan profesi
2. **About Me** - Deskripsi diri, skills dengan progress bar, pengalaman
3. **Certificates** - Daftar sertifikat dalam card layout

## Teknologi
- PHP (MySQLi)
- MySQL
- CSS3
- Bootstrap 5

## Cara Menjalankan
1. Import file `database.sql` ke MySQL.
2. Atur kredensial database di `config.php` sesuai lokal Anda.
3. Jalankan project melalui server lokal (contoh: XAMPP atau Laragon).
4. Buka `index.php` di browser.
