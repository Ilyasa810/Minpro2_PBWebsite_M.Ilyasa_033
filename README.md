## 🚀 Cara Menjalankan Project

Project ini dijalankan menggunakan **XAMPP (Apache Server)**.

### 1️⃣ Install XAMPP
Download dan install XAMPP jika belum ada.

### 2️⃣ Pindahkan Project ke Folder htdocs

Salin folder project ke dalam directory berikut:

```
xampp/htdocs/
```

Contoh:

```
xampp/htdocs/MINPRO 2
```

### 3️⃣ Jalankan Apache

1. Buka **XAMPP Control Panel**
2. Klik **Start** pada **Apache**

### 4️⃣ Buka Website di Browser

Akses website melalui browser dengan alamat berikut:

```
http://localhost/MINPRO%202/index.php/
```

Website akan otomatis terbuka melalui **local server**.
